Sales Dashboard

Description
This dashboard visualizes sales data to provide insights on revenue, product performance, and sales trends over time. It helps track key metrics and supports decision-making for the sales team.

Features
- Interactive charts for monthly and yearly sales
- Top-selling products and regions
- Trend analysis and comparisons
Tools
Tableau Public

Accident Road Dashboard

Description
This dashboard analyzes road accident data to identify patterns, high-risk areas, and contributing factors. It supports traffic safety management and helps in planning preventive measures.

Data Source
- Dataset includes accident location, date, severity, and type of vehicle involved.

 Features
- Accident trends over time
- Breakdown by severity and type

Tools
Tableau Public
